# ✍️ Signature Animator

A web app that turns your signature into a smooth, animated SVG — draw it live or upload an image.

## Features

- **Draw** — Freehand canvas with pressure-sensitive, variable-width strokes
- **Upload** — Drop in a signature image (PNG/JPG/SVG) for automatic SVG tracing
- **Animation Styles** — Letter by Letter, Continuous Flow, or Classic playback
- **Speed Control** — Adjustable animation speed slider
- **Export** — Download SVG/PNG, plus animated GIF and MP4 outputs
- **Reliability Fallback** — Automatically switches to server-side export when browser encoding fails

## Tech Stack

React · Vite · Framer Motion · Tailwind CSS · Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
